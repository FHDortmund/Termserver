Terminologieserver - Release

Webservices


Einen Überblick über alle Webservices finden Sie auf der Seite Webservices Terminologieserver.
Dort sind zudem alle WSDL-Datreien aufgelistet, welche für die Implementierung von Webservices nötig sind.

Folgende Dienste werden angeboten:
Administration (Administration)
Link http://193.25.22.68:8080/Terminologieserver/Administration?wsdl
Methoden Vokabular importieren (Import Code System)
Vokabular exportieren (Export Code System)
Authorization (Authentifizierung)
Link http://193.25.22.68:8080/Terminologieserver/Authorization?wsdl
Methoden Einloggen (Login)
Ausloggen (Logout)
Authoring (Pflege)
Link http://193.25.22.68:8080/Terminologieserver/Authoring?wsdl
Methoden Vokabular erstellen (Create Code System)
Vokabular bearbeiten (Maintain Code System Version)
Status eines Vokabulars ändern (Update Code System Version Status)
Begriff erstellen (Create Concept)
Begriff bearbeiten (Maintain Concept)
Status eines Begriffs ändern (Update Concept Status)
ValueSet erstellen (Create ValueSet)
ValueSet bearbeiten (Maintain ValueSet)
Status eines ValueSets ändern (Update ValueSet Status)
Concept Association (Konzept-Verbindungen)
Link http://193.25.22.68:8080/Terminologieserver/Association?wsdl
Methoden Verbindungen zwischen Begriffen auflisten (List Concept Associations)
Details zu Verbindungen erhalten (Return Concept Association Details)
Verbindungen erstellen (Create Concept Association)
Verbindungen bearbeiten (Maintain Concept Association)
Search (Suche/Zugriff)
Link http://193.25.22.68:8080/Terminologieserver/Search?wsdl
Methoden Vokabulare auflisten (List Code Systems)
Details zu Vokabularen erhalten (Return Code System Details)
Begriffe auflisten (List Code System Concepts)
Details zu Begriffen erhalten (Return Concept Details)
Verbindungstypen auflisten (List Concept Association Types)
Details zu Verbindungstypen erhalten (Return Concept Ass. Type Details)
ValueSets auflisten (List ValueSets)
Details zu ValueSets erhalten (Return ValueSets Details)
Inhalte zu ValueSets auflisten (List ValueSet Contents)
